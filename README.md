# YOLO Real-Time Alert Project

This repository contains two main computer vision scripts:

- **alarm_example.py**: Real-time YOLOv8 detection with sound alerts.
- **yolo_cam_realtime.py**: Real-time YOLOv8 detection without sound alerts.

## Project Structure
- `src/` — source code  
- `tests/` — testing files  
- `sample/` — sample input files  

Run the project:
```
make run
```
